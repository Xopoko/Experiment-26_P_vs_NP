# Открытые вопросы (рабочая очередь)

Правило: **каждый прогон агента выбирает ровно 1 пункт ниже** и делает измеримый прогресс:
доказательство/контрпример/уточнение формулировки/точная ссылка + барьер‑чек. Затем обновляет пункт.

## Активные

- [ ] **Q1 (Pich–Santhanam 2023):** формально выписать определение `tt(h_{0,n},2^{n/4},t(n))` и где именно оно используется в выводе (см. `docs/15_proof_complexity.md`).
- [ ] **Q2 (EF+assumptions ⇒ P≠NP):** распаковать “две предпосылки S$^1_2$” в виде конкретных $\Pi^b_1$‑формул (что именно должно быть доказуемо) и проверить, где возможны «скрытые» усиления.
- [ ] **Q3 (минимальный нерелятивизирующий шаг):** найти/сформулировать лемму, которая *по определению* не релятивизируется (и реально нужна выбранной линии атаки), и довести её до проверяемого состояния (или честно зафейлить с причиной).
- [ ] **Q4 (канонизация/ROABP‑барьер):** уточнить классы входных сертификатов (IPS/EF‑кодировки), для которых “эффективная multilinearization → ROABP” была бы уже достаточно сильной, и записать самый слабый корректный барьер‑вариант.
