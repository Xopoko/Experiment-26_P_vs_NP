# Q43.S324 nk base k0

We define the explicit threshold `k0(C)=max(13, 2*(2*C*c1))` and show that
`k>=k0(C)` implies `3*(2*C*c1) <= 2k+1`, hence the base bound
`2*C*c1 <= 2^(2k+1)/(2k+1)^5`. This is wired into the gap-right lift so the
`n>=n_k+1` interval can be discharged once k is fixed.
