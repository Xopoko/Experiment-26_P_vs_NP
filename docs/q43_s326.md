# Q43.S326 gap-right global n0

We define `n0(C)=3*2^(k0(C)-1)` and prove a wrapper lemma that applies the
k0-based gap-right lift directly from the bound `n0(C) <= n` plus
`n < 2^(k0(C)+1)`. This packages the k0 dependency into a single threshold
for the upper-plateau regime and is the entry point for threading into the
regime-d chain.
